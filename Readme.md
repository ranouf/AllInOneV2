# AllInOne

This project is a base for future projects. It will include:
- A .Net Core 3.0 API with:
	- Authentication with JWT
	- Sql Server Database
	- Entity Framework
	- AutoMapper
	- Dependency Injection using Autofac
	- Swagger
	- Miniprofiler
	
- Test Integration to test the API

- An Angular App

**Project in progress**
